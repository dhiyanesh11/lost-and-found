# lostandfound

### To run the application 
1.clone the repo
```bash
git https://github.com/dhiyanesh11/lost-and-found.git

```
2.Open in the terminal
3. Run the Server
```bash
cd server
npm install
npm start
```

4.Run the Client-side application
```bash
cd client
npm install
npm run dev
```


## Features
- Student & Admin roles
- JWT Authentication
- AWS S3 image storage
- Found item claim workflow
- Admin claim approval system

## Tech Stack
- React
- Node.js
- Express
- MongoDB
- AWS S3
